# Project Details

## Project Name: 
- Kenkeeper

## Project Description:
- This is a simple relationship management web application where users can explore and track interactions with their friends.

- On the homepage, all friends are displayed in card format. Clicking on any card navigates to a detailed view of that specific friend. In the details page, users can log interactions using three options: Text, Call, and Video.

- Each interaction is recorded in a timeline, where users can:

- View all activities
- Filter by interaction type (Text, Call, Video)
- Search by friend name

- Additionally, the app includes a chart page that visually represents the number of interactions based on their types (Text, Call, Video),      helping users quickly understand their communication patterns.

## 🚀 Technologies Used

- 🧱 **HTML5** — Structure of the application  
- 🎨 **Tailwind CSS** — Modern and responsive UI design  
- ⚛️ **React.js** — Component-based frontend development  
- ▲ **Next.js** — Full-stack framework with routing and optimization  

## 🚀 Key Features
- 🎴 Interactive Friend Cards
- 📊 Smart Timeline Management
- 📈 Visual Insights with Charts
